Privacy Policy

Claude RTL — Chrome Extension  |  Last updated: March 2026

In short: Claude RTL does not collect, store, transmit, or share any user data whatsoever. Everything happens locally in your browser.

What data does this extension access?

Claude RTL only accesses the DOM (Document Object Model) of pages on claude.ai in order to apply right-to-left text direction styling. It does not read, collect, or transmit any message content, personal information, or browsing data.

Data storage

The extension stores a single boolean preference (RTL on/off) using Chrome's local storage API (chrome.storage.local). This value:

  • Never leaves your device
  • Is not sent to any server
  • Is not shared with any third party
  • Can be cleared by removing the extension

Permissions explained

  • Host permission (claude.ai) — Required to inject RTL styles into Claude.ai pages.
  • storage — Required to remember your on/off preference between sessions.

Third-party services

This extension does not use any third-party analytics, tracking, advertising, or data collection services.

Network requests

This extension makes zero network requests. It operates entirely offline within your browser.

Changes to this policy

If this privacy policy is ever updated, the changes will be reflected on this page with an updated date.
